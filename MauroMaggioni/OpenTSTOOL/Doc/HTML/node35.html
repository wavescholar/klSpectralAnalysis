5.3.8 Help

The menu USAGE will start your web browser with the HTML-Version of this manual shipped with the OpenTSTOOL distribution. Run 'help docopt' at the matlab command prompt to configure your web brower correctly.

After the seperation line you can view the matlab command line help for every method of the signal class. The same information is in the HTML- and in the PDF-Version of the manual in class reference section 6.20.
