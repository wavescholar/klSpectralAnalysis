4. Nearest Neighbors Searching

An integral part of a majority of methods for nonlinear time series analysis is searching for nearest neighbors. The perfomance of these methods depends strongly of the perfomance of the employed nearest neighbor algorithm. Thus, choosing an efficient nearest neighbor algorithm should be done very carefully.
