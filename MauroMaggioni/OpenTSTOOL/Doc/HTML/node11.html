2.4 Pitfalls

See also the FAQ (Frequently Asked Questions) 7!

  1. As of TSTOOL version 1.2, the main function for creating the TSTOOL GUI is called opentstool instead of tstool to avoid collision with the time series toolbox from The Mathworks.
  2. When using Winzip, enable Use path information to make sure that subdirectories are created.
  3. TSTOOL will not work with Matlab version prior to 6.5!
  4. It's not a good idea to place the TSTOOL distribution into the Matlab directory. We obtained reports about strange bugs occuring when the TSTOOL distribution is extracted into the directory where the Matlab system is installed.
